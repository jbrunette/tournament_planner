Tournament Planner
=============================

INSTRUCTIONS
====================

1. Install postgresql and python onto your computer.
2. Access postgresql by running psql on the command line.
3. Create database "tournament" in postgresql with the command "CREATE DATABASE tournament;"
4. Connect to the database via the command "\c tournament"
5. Create the required tables by running the command "\i tournament.sql"
6. Exit psql by running the command "\q"
7. Run tournament.py via python to ensure code works properly.
